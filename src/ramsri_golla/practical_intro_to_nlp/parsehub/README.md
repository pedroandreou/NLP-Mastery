## Download Parsehub
```
curl -L https://parsehub.com/static/client/parsehub.tar.gz | tar -xzf - -C /tmp && \
sudo mv /tmp/parsehub /opt/ && \
sudo ln -s /opt/parsehub/parsehub /usr/local/bin/
```
